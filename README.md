# Ashwood County 3D Prototype

This repository is a technical feasibility prototype for a possible 3D version of Ashwood County.

It exists to answer three questions:

- Can a visually appealing third-person survival game be built in Godot?
- Can it be developed comfortably on integrated graphics?
- Can modern AI coding agents dramatically accelerate development?

## Technology

- Godot 4.7.1 .NET
- C#
- Compatibility Renderer

## Current Status

Prototype

## Goals

- Third-person movement
- Third-person camera
- Simple environment
- Basic lighting
- One zombie
- Performance evaluation

## Non-goals

This repository is **not** intended to become the full game.

Large gameplay systems such as:

- inventory
- combat
- quests
- AI
- dialogue
- saving

are intentionally excluded.

The purpose is to evaluate whether a full 3D version of Ashwood County is technically and creatively viable.

## Development

The project is AI-assisted using Cursor and Codex.

Every feature should be implemented as a small, independently testable vertical slice.
